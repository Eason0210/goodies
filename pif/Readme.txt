File imtools_pif.h and import library imtools.lib
provide a set of routines for specifying and writing
PIF files. 

File imtools_pif_example.c shows how to use the PIF 
specification routines contained in the DLL. File 
imtools_pif_example.vcxproj will help you build the 
example program in the Microsoft's Visual C++ 2010 
environment.

Please note that you will need to copy the file 
imtools.dll in the directory where will be located
your run-time executable.

For additional information, please contact InnovMetric.
